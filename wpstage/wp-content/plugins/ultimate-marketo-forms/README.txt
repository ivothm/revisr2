The Ultimate Marketo Forms for WordPress plugin enables auto-fill, progressive profiling, custom confirmation page logic, and more with a simple install.

== Description ==

Implementing full-featured Marketo lead forms on your WordPress website used to require days of expert-level custom coding. The Ultimate Marketo Forms for WordPress plugin contains the following features:

* Global forms admin using shortcodes
* Multiple forms per page
* Auto pre-fill form fields
* Progressive profiling
* Custom thank you url
* Add url variables to custom hidden fields
* Use css to update marketo form styles

<a href="http://www.hyperxmedia.com/ultimate-marketo-forms-wordpress-plugin/" target="_blank">Click here</a> to learn more.


== Installation ==

How to install the plugin and get it working.

1. Upload `ult_marketo_forms` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Marketo API details
4. View the list of forms in your Market instance and customize the form by clicking on it’s name.
5. To use the form, simply copy the shortcode next to the form name and paste it into any page or post.


== Frequently Asked Questions ==

= Where do I find my Marketo API Settings? =

Marketo has provided a quick tutorial on how to obtain the Endpoint, Client ID and Client Secret needed for the plugin to connect to your account here: http://developers.marketo.com/rest-api/

= How do I unlock the full features of the plugin? =

Simply complete the form on our site to be emailed your custom token to unlock the plugin.  You can find the form here: http://www.hyperxmedia.com/ultimate-marketo-forms-wordpress-plugin/
